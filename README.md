# HerokuLifx

Shows heroku status through lifx bulbs.


Tired of waiting for the june 23, 2014 issue to resolve, I've set my LIFX
bulb to wait for it ...

## Installation

    $ gem install heroku_lifx

## Usage

    $ heroku_lifx
